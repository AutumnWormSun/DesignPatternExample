■『Design Patterns於Java語言之實習應用』 隨附光碟之注意事項（編輯部）

　感謝讀者購買『Design Patterns於Java語言之實習應用』一書。
以下介紹本光碟內容以及使用時的注意事項，請於使用本光碟前詳細閱讀。

●光碟內容

　中文化之後的範例檔放在 [chinese] 目錄下，並以章節來區分子目錄，
書中範例都放入sample目錄，而Q和A開頭的目錄分別為問題與解答相關程式檔案。
如果讀者環境有支援日文，想取用原書作者日文範例，可讀取 [ japan ] 目錄，
其中適用於Windows系統的程式範例在[Windows]目錄底下，適用於UNIX系統的程
式範例則放在[Unix]目錄下。

　各目錄的意義如下：

　　src/Pattern名稱/Sample      Pattern章節中解說的範例程式
　　src/Pattern名稱/Q☆...      Pattern章節的練習問題中出現的LIST（☆是問題編號）
　　src/Pattern名稱/A☆...      Pattern章節的練習問題的解答中出現的LIST（☆是問題編號）
